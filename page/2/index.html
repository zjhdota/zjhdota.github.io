xps13 安装Kali Linux

发表于 2017-03-13 | 分类于 Kali Linux

准备

  • Dell XPS 13-9360
  • 支持 UEFI的至少8G的U盘
  • 官网下载镜像文件kali 官网
  • Window系统下载 Win32 Disk Imager
  • 用Win32 Disk Imager 制作U盘启动盘

开始

  • 开机 进入按 F2 进入 bios 选择general->boot sequence 模式改为uefi,在secure boot->secure boot enable 中选择 disable，最后在systerm configuration->SATA operation选择 AHCI，保存并重启
  • 重启后按F12进入bios，检查启动模式是否是UEFI secure boot 是否是off 若是则跳到3步骤，不是则在other options里选择change boot mode settings 把启动方式改为UEFI secure boot改为 off
  • 重启后按 F12，在UEFI boot下选择U盘启动（这样目的是安装GPT分区）。
  • 选择图形界面安装。
  • 下一步直到分区，选择自动分区将/home 用户单独分区，下一步直到安装完成。
  • 拔掉U盘重启，按F2 进入BIOS，在general->boot sequence选项卡中 手动添加efi文件EFI->kali->grubbx64.efi，配置好后重启。
  • 成功进入kali linux
